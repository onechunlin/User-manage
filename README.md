# 用户管理系统

## 注意事项
因为github大小限制的原因，node_modules文件夹无法上传，如果需要运行完整的项目，请先使用Vue脚手架创建一个项目，然后将node_modules文件夹拷过来即可。由于需要后台接口的配合，所以得自行搭建jsonserver的后台接口，搭建教程可自行百度，记得最后根据后台地址修改前端数据请求处的地址即可。

## 项目实现
采用Vue框架搭建的前端，后台数据接口通过jsonserver搭建
